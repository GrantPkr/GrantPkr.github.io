Project Highlights

This section highlights technical work that reflects my interests in AI, machine learning, software engineering, and intelligent decision-making systems.

Capstone Project: Agentic Frontend Generation System

One of the most important projects I have worked on is my capstone project involving an agentic system designed to generate production-aligned frontends. What makes this project valuable to me is that it exists within real constraints rather than as a purely classroom exercise. It required thinking about standards, workflows, and the gap between generated output and usable software. This project helped me better understand collaboration, software quality, and how intelligent systems need to be guided in order to produce results that are actually helpful in practice.

AI/ML Trading Project

Another project I want to highlight is my AI and machine learning trading project. This work focused on designing a system that used algorithmic and data-driven reasoning in a market-oriented setting. What I value most about this project is that it combined technical implementation with evaluation and interpretation. It pushed me to think more carefully about methods, assumptions, and how to present technical results clearly. Mancala AI with Minimax and Alpha-Beta Pruning

View Repository

My Mancala AI project is another good example of the kind of work I enjoy. In that project, I implemented decision-making strategies using minimax and alpha-beta pruning, which gave me hands-on experience with evaluation functions, search depth tradeoffs, and algorithmic reasoning. I like this project because it demonstrates a more classic AI perspective and shows how abstract ideas become concrete when you have to make a system choose effectively under constraints. It also helped strengthen my understanding of strategy, efficiency, and explainable logic in intelligent systems.
